### Configurar Entorno Python

Activar el entorno virtual (macOS/Linux):
```bash
python3 -m venv venv
```
```bash 
source venv/bin/activate
```

Windows:
```bash
python -m venv venv
```
```bash
venv\Scripts\activate
```

### Instalar dependencias:
```bash
python -m pip install -r requirements.txt
```
